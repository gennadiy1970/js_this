This

  • this как указатель на собственные свойства/методы объекта
  • this в глобальном объекте
  • this в объекте
  • new и конструктор объекта
  • bind это способ для метода переопределить this объекта
  • call & apply это способы для объекта использовать функцию как собственный метод
  • this и стрелочные функции

    • Правила this

    • значение this, внутри функции, определятся по месту вызова функции
    • значение this, внутри стрелочной функции, определятся по месту создания стрелочной функции
