Объект

this внутри объекта это ссылка на сам этот объект:

const parent = {
    method: function () {
       return  this === parent;
    }
};
                
parent.method(); // true

Объект можжет быть вложенным, тогда и this указывает на вложенность:

const grandparent = {
    parent: {
        method: function () {
            return  this === grandparent.parent;
        }
    }
};
                        
grandparent.parent.method(); // true

А т.к. все объекты скрипта браузера находятся внутри объекта window, то 

window.grandparent.parent.method(); // true
